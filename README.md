# Python To-Do List (CLI)

A simple command-line To-Do List application written in Python.
This program allows users to add, view, and remove tasks directly from the terminal.

## Features

* Add new tasks
* View all tasks with numbering
* Remove tasks by selecting their number
* Input validation to prevent invalid entries
* Colored terminal interface using Colorama

## How It Works

1. The program displays a menu with options.
2. Users can choose to add, view, or remove tasks.
3. Tasks are stored in a list during the program execution.
4. Users can remove tasks by entering the task number.

## Example

```
Welcome To To Do List

1. Add Task
2. View Tasks
3. Remove Task
4. Exit

Enter your choice (1-4): 1
Enter the task you want to add: Learn Python

Task added successfully!

Enter your choice (1-4): 2

Tasks in the list:
-----------
1. Learn Python
```

## Requirements

Python 3.x
Colorama library

Install Colorama using:

pip install colorama

## Run the Program

python to-do-list.py
